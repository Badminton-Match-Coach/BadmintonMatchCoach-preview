Privacy Policy

Badminton Match Coach respects your privacy. This policy explains what data the Android app collects, how it is stored, and how it is used.

Data collected

  • Match event details entered by users, including rally outcomes, shot types, and scoring events — kept in the app's runtime memory by default.
  • Match metadata such as player names, competition details, and session timestamps — kept in the app's runtime memory by default.
  • Local session state used to recover match progress while the app is running.
  • Optional .BmMatch files opened or exported by the user. When you choose to save or export a match file, it is written to the device where the app is running (by default to the Documents/Badminton folder) and not uploaded or shared automatically.

How data is used

Match data is used only to provide analysis, scoring, undo/redo history, and report generation. Data remains local unless you choose to export a .BmMatch file or share a report.

Storage and offline use

The app keeps match data in its runtime memory while you use it. If you explicitly save or export a match (for example to a .BmMatch file), the file is written to the device's storage — by default we place exported files in the Documents/Badminton folder on the device where the app is running. The app operates fully offline and does not upload data unless you manually share or export a file.

Permissions

No location, camera, or personal account permissions are required. The app only requests storage access when saving or opening a match file.
