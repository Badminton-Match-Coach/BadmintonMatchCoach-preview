About this app

Badminton Match Coach is a professional badminton match analyzer built for real matches and post-game review. It supports BWF-standard scoring, .BmMatch match records, and automatic local persistence so sessions are recovered after refresh or reload.

Fast scoring

Tap each rally serve and end with, final shot, and outcome details to capture the full match flow.

BWF rules

Tracks games to 15 with a 2-point margin, max 21, and standard match progression for singles and doubles.

Match recovery

Automatic local storage sync preserves your session and allows seamless resume after reopening the app.

Don't forget to update games, score and server information (see Resume screen above).

How it works

  1. Start a new match — use the quick new match shortcut to open a session instantly.
  2. Track every rally — log shots, outcomes, and service position in real time.
  3. Open or save .BmMatch files — load saved match recordings or export the analysis for later review.

Match analysis

Analyze every match with built-in post-game summaries, rally histories, and export-ready reports. The app is designed to help you review performance and improve strategy at every level.

Heat maps

See where points were won and lost on the court for better tactical planning.

Shot statistics

Measure smash accuracy, net play success, drop shots, clears, and more.

Rally timeline

Open any rally and review how the point developed shot by shot.

File handling

Load and save .BmMatch files to preserve match analysis and collaborate with coaches and teammates.

System requirements

Badminton Match Coach is a standalone Android app built for real match recording and analysis.

  • Android 5.0 (API level 21) or higher
  • Target API level 33
  • Supports opening and handling .BmMatch match files via Android file and content intents
  • Includes a quick "New Match" shortcut for fast match setup
  • Requires standard Android app permissions only; it runs natively on Android as a standalone app
